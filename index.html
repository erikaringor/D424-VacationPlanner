Vacation Planner Full-Stack Application

This Android mobile application was developed for the D424 Software Engineering Capstone. It allows users to manage vacations and excursions, search vacation records, generate reports, and use login validation.

Application Features

  • Login validation
  • Vacation and excursion management
  • Search functionality
  • Vacation report with timestamp
